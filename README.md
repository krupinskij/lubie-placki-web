## Installation

Download or clone repository

Add .env file

```
REACT_APP_API_URL=...
REACT_APP_TOKEN_KEY=...
PORT=...
```

Run

```bash
# installing dependances
$ yarn install
```

## Running the app

```bash
# development
$ yarn start
```
